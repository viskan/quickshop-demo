# Streamline Express Demo

## Build for production

Host the `public` directory on a server of your choice.

## Docker

$ docker build --tag viskan/streamline-express-demo:latest .

## Preview in localhost

Make sure you have [Node.js](https://nodejs.org/en/) installed before doing these steps.

```
$ npm install
$ npm run dev
```

Alternatively, you can use any server that supports serving static files, and point it to the `public` directory.
